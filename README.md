## 學習老師優點

> 故意將名稱隱藏

1. 老師會將程式碼放在`Dropbox`存檔自動同步，漏掉可以馬上比對。問問題會請大家在聊天室互動(怕大家害羞)。每節課的程式碼依照順序排序，辨識度高。
2. 老師會分享工作經驗例如:撰寫`Function`前面加上判斷，畢竟不能保證其他人會更動執行順序。很認同老師上課說的一句話:"你不是學生了，要把自己當成工程師來分析並解決問題"
3. 老師不斷提醒我們軟體工具要以邏輯方式去熟練操作，不是死記步驟。
