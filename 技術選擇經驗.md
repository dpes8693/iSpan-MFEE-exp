## HTML

### 什麼時候綁定事件

```html
<!-- 這樣不用提早綁定 也比較安全(方法名稱不會暴露在html內被user看到)  -->

<button id="btn">什麼時候綁定</button>
<script>
  btn.onclick = function () {
    console.log("後面綁");
  };
</script>

//這樣好處不用在js綁定元素比較乾淨
<button onclick="f()">什麼時候綁定</button>
<script>
  function f() {
    console.log("前面綁");
  }
</script>
```
