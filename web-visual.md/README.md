## 作業

``` md
製作主題：
    - 個人簡歷figma製作(網頁及手機版)
      - 一張PSD 圖片（此圖片可為figma 裡面需要用的圖片）
      - 內容至少三區塊

    - 製作尺寸
      - PSD不拘 解析度72即可
      - FIGMA：
            桌機  Ｗ：1440px    Ｈ：不限
            手機   Ｗ：375px      Ｈ：不限
```
